# Adv_Chatbot_with_sidbar
AI-powered chatbot built using LangChain, Groq, and Streamlit with memory and translation support.
This project is an AI-powered chatbot built using LangChain, Groq API, and Streamlit. It provides fast and intelligent responses with memory support to maintain conversation history. The chatbot also includes features like language detection and translation, making it useful for multilingual interactions. The goal of this project is to create a simple, efficient, and interactive chat application with a clean user interface
